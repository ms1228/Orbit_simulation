This routine demonstrates the construction of a pseudo two dimensional array, abc, by concatinating three 1-dimensional "row arrays", a,b, and c.

This prints the value of one variable (a2):

array demo


Now print the values of one 1-dimensional "row" array (a):

array demo 1


Now print the values of the complete two dimensional array abc = [a,b,c]:

array demo 2


Finally print each "row array" separately:

array row 1

array row 2

array row 3
